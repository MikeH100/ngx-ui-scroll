In case the data is accessible via some remote API, the Datasource.get could be implemented using Observable signature. In this demo we have a nodejs application with /api/data GET request handler ("Server" tab), which emulates a remote limited Datasource implementation. The Component just provides /api/data GET method call inside Datasource.get body. And again, the only requirement is that the API handler on server must return an array of index-count items or empty array if we are out of range.
